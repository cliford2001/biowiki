Integración de datos proteómicos y transcriptómicos con SCENIC y AUCell en células madre

Introducción: El poder de la integración multivariada en biología celular

Las células madre representan un campo clave en biología celular y regenerativa, donde la comprensión de su estado de pluripotencia y sus vías de diferenciación requiere un análisis integral de múltiples capas moleculares. Aunque el análisis transcriptómico mediante secuenciación de RNA (RNA-seq) ha sido el estándar durante años, la integración con datos proteómicos aporta una dimensión crítica: la función y regulación proteica, que puede no correlacionarse directamente con los niveles de mRNA debido a la regulación post-transcripcional.

En este contexto, herramientas como SCENIC (Single-Cell Regulatory Network Inference and Clustering) y AUCell (AUCell: a fast algorithm for calculating enrichment scores in single-cell data) se han posicionado como pilares para el análisis integrado. SCENIC permite reconstruir redes regulatorias a partir de datos transcriptómicos, identificando reguladores transcripcionales activos en cada célula. AUCell, por su parte, permite calcular puntuaciones de activación de estas regulaciones en un conjunto de datos, incluso cuando no se dispone de datos proteómicos directos. Esta combinación permite inferir la actividad de redes reguladoras de manera robusta, aportando una visión funcional más completa.

Desarrollo: Estrategia para la integración de datos

1. Preprocesamiento de datos transcriptómicos y proteómicos

El primer paso es asegurar que ambos conjuntos de datos (transcriptómico y proteómico) estén correctamente preprocesados y alineados en el mismo espacio de células. Para los datos transcriptómicos, se recomienda usar herramientas como kallisto o salmon para la cuantificación de expresión, seguido del uso de Seurat o Scanpy para el análisis de datos de secuenciación de célula única (scRNA-seq).

Para los datos proteómicos, técnicas como mass spectrometry (MS) generan datos de abundancia proteica. Estos deben ser normalizados y convertidos en un formato compatible con los datos transcriptómicos, a menudo mediante la creación de una matriz de expresión proteica donde las filas son proteínas y las columnas son células. Es crucial que ambas matrices compartan la misma etiqueta de célula (cell ID) para poder fusionarlas adecuadamente.

2. Construcción de redes regulatorias con SCENIC

SCENIC utiliza una combinación de análisis de coexpresión y predicción de sitios de unión para inferir redes regulatorias. El proceso comienza con la identificación de módulos de coexpresión (gene modules) a partir de los datos transcriptómicos. Luego, se utiliza un modelo de regresión para predecir qué factores de transcripción (TFs) podrían estar regulando esos módulos.

La herramienta scenic (implementada en R) permite ejecutar este pipeline completo. A continuación se muestra un fragmento de código R para construir la red regulatoria:

library(SCENIC)

# Cargar datos de expresión (exp_matrix) y metadata (meta)
# exp_matrix: gene x cell, con genes como filas y células como columnas
# meta: información adicional sobre las células

# Construir el objeto SCENIC
scenic_obj <- createSCENICobject(
  expressionMatrix = exp_matrix,
  metadata = meta,
  species = "human"  # o "mouse" según el organismo
)

# Calcular reguladores transcripcionales activos (GRN)
scenic_obj <- runSCENIC(scenic_obj, nCores = 8)

3. Evaluación de la activación de reguladores con AUCell

Una vez construida la red regulatoria con SCENIC, se utiliza AUCell para calcular las puntuaciones de activación de los reguladores transcripcionales en cada célula. Esto permite identificar qué células están activando qué redes regulatorias, incluso si no se tiene acceso directo a datos proteómicos.

A continuación, un ejemplo de cómo se ejecuta AUCell en Python:

import scanpy as sc
import pandas as pd
from auctool import aucell

# Cargar el archivo de genes de reguladores (GRN) generado por SCENIC
grn_df = pd.read_csv("grn.csv", index_col=0)

# Cargar el objeto de expresión (AnnData)
adata = sc.read("expression_data.h5ad")

# Calcular puntuaciones con AUCell
auc_scores = auctool.aucell(
    adata.X,  # matriz de expresión
    grn_df,   # GRN: TF x genes
    min_n_genes=5,
    n_jobs=8
)

# Añadir puntuaciones al anndata
adata.obs["auc_scores"] = auc_scores

4. Integración con datos proteómicos

El paso final es la integración de las puntuaciones de activación de reguladores (obtenidas por AUCell) con los datos proteómicos. Esto puede hacerse mediante análisis correlacionales entre las puntuaciones de AUCell y los niveles proteicos, o mediante métodos más avanzados como multi-omics integration en Seurat o Scanpy.

Por ejemplo, se puede usar una regresión para predecir la abundancia proteica a partir de la activación de reguladores transcriptómicos. Esto permite inferir cómo las redes reguladoras transcriptómicas influyen en la actividad proteica en células madre.

Resumen

La integración de datos proteómicos y transcriptómicos mediante SCENIC y AUCell ofrece una visión profunda de la regulación celular en células madre. SCENIC permite inferir redes regulatorias activas a partir de datos transcriptómicos, mientras que AUCell proporciona una estimación robusta de la activación de esas redes en cada célula. Al combinar estos resultados con datos proteómicos, se puede explorar la concordancia entre regulación transcriptómica y proteica, identificar reguladores clave en la pluripotencia y la diferenciación, y descubrir nuevas rutas funcionales. Esta estrategia es especialmente poderosa en escenarios donde los datos proteómicos son escasos o incompletos.

Errores comunes y buenas prácticas

  • Usar datos sin normalización adecuada: Asegúrate de normalizar tanto los datos transcriptómicos como los proteómicos antes de cualquier análisis. El uso de datos sin normalizar puede generar sesgos significativos.
  • Ignorar la calidad de los datos: Evalúa la calidad de los datos de secuenciación y de espectrometría de masas. Filtra células de baja calidad o con alto contenido mitocondrial.
  • Suponer correlaciones directas: La expresión de mRNA no siempre se traduce en proteína. Asegúrate de interpretar las correlaciones entre AUCell y proteínas con cautela, y considera factores de regulación post-transcripcional.
  • Usar un número insuficiente de células: Para construir redes regulatorias confiables, necesitas un número suficiente de células (generalmente > 500).
  • No validar los resultados: Comprueba la biología detrás de los reguladores identificados. Usa herramientas como Enrichr o Gene Ontology para validar si los genes en un módulo están biológicamente relacionados.
