Anotación de tipos celulares en scRNA-seq

Introducción

La anotación de tipos celulares en datos de secuenciación de ARN de células únicas (single-cell RNA sequencing, scRNA-seq) es un paso crítico en el análisis de transcripción a nivel celular. Tras la obtención de perfiles de expresión génica de miles de células individuales, el investigador se enfrenta a un reto fundamental: ¿qué tipo de célula representa cada cluster identificado? Esta anotación no es solo un ejercicio de etiquetado, sino una puerta hacia la comprensión funcional del tejido, la identificación de subpoblaciones raras, la caracterización de estados celulares durante el desarrollo o la respuesta a tratamientos, y la generación de hipótesis biológicas.

La precisión de la anotación determina la validez de las conclusiones derivadas del estudio. Una etiqueta incorrecta puede llevar a interpretaciones erróneas sobre la biología celular, la dinámica de tejidos o los mecanismos de enfermedad. Por ello, se requieren estrategias robustas, basadas en marcadores génicos conocidos y técnicas estadísticas que permitan una asignación confiable.

Metodologías de anotación

Existen varios enfoques para anotar tipos celulares, cada uno con sus fortalezas y limitaciones. A continuación se describen los métodos más comunes.

1. Anotación basada en marcadores conocidos

Este método consiste en buscar genes que están altamente expresados en un cluster y que son conocidos por marcar tipos celulares específicos. Se utilizan bases de datos como Human Protein Atlas, Mouse Phenotype, IHC Web, o listas manuales de marcadores de literatura. Por ejemplo:


# En R con Seurat
marker_genes <- GetAssayData(sce, slot = "data")

# Buscar marcadores para linfocitos T
T_cell_markers <- c("CD3D", "CD3E", "CD4", "CD8A", "TRAC")

# Ver expresión promedio en cada cluster
AverageExpression(sce, features = T_cell_markers)

Este enfoque es directo y rápido, pero depende de la calidad y exhaustividad de los marcadores disponibles. Puede fallar si se trata de una población celular poco estudiada o si hay expresión inusual de marcadores.

2. Anotación por comparación con referencias públicas

Este enfoque implica comparar los perfiles de expresión de los clusters con conjuntos de datos de referencia ya anotados. Herramientas como SingleCellMultiModal, CellTypist, o scRNA-seq-analyzer permiten clasificar células mediante modelos de aprendizaje automático entrenados en datos de referencia. Por ejemplo, CellTypist utiliza un modelo de árboles de decisión para predecir el tipo celular basado en la expresión de genes de referencia.


# En Python con CellTypist
import celltypist

# Cargar modelo
model = celltypist.load_model("celltypist_model")

# Predecir tipos celulares
predictions = model.predict(data=cluster_expression, return_prob=True)

Esta metodología es más robusta que la basada en marcadores únicos, ya que considera patrones complejos de expresión y puede identificar poblaciones no esperadas. Es especialmente útil en tejidos con alta heterogeneidad o en sistemas modelo menos estudiados.

3. Anotación guiada por análisis de funciones

En casos donde la identidad celular no es clara, se puede recurrir al análisis funcional. Se identifican genes diferencialmente expresados (DEGs) entre clusters y se analiza su enriquecimiento funcional mediante herramientas como Gene Ontology (GO), Reactome, o GSEA. Por ejemplo, si un cluster muestra un enriquecimiento significativo en vías relacionadas con la actividad mitocondrial, podría corresponder a una población de células de alto metabolismo como los macrófagos activados.


# En R con Seurat
# Encontrar DEGs entre cluster 1 y 2
degs <- FindMarkers(sce, ident.1 = 1, ident.2 = 2, test.use = "wilcox")

# Enrichment con clusterProfiler
library(clusterProfiler)
library(org.Hs.eg.db)

ego <- enricher(degs$avg_log2FC, 
                gene = rownames(degs), 
                universe = rownames(degs), 
                pvalueCutoff = 0.05, 
                pAdjustMethod = "BH")

Este método no proporciona una etiqueta directa, pero ayuda a inferir la función biológica del cluster, lo que puede facilitar la anotación mediante lógica biológica.

Resumen

La anotación de tipos celulares en scRNA-seq es un paso esencial para extraer significado biológico de los datos. No existe un único método perfecto. La mejor estrategia combina múltiples enfoques: se comienza con marcadores conocidos, se valida con modelos de referencia, y se refuerza con análisis funcional. La anotación cuidadosa y justificada es clave para que los hallazgos sean reproducibles y biológicamente relevantes.

Errores comunes y buenas prácticas

  • No confiar únicamente en un marcador único: los marcadores son a menudo expresados en múltiples tipos celulares. Usar múltiples marcadores específicos mejora la confiabilidad.
  • Ignorar la calidad de los datos: un cluster con baja cobertura o alta contaminación puede llevar a anotaciones erróneas. Verificar la calidad previamente (por ejemplo, con scater o Scanpy).
  • Usar referencias no relevantes: los datos de referencia deben ser del mismo organismo, tejido o estado biológico. Comparar un cluster de corazón con datos de cerebro puede dar resultados falsos.
  • Rechazar la evidencia funcional: si un cluster no se alinea con marcadores conocidos, pero tiene un perfil funcional coherente, puede representar una población nueva o una condición poco común.
  • Documentar el proceso: registrar cómo se llegó a cada anotación (marcadores usados, referencias consultadas, resultados de enriquecimiento) permite la revisión y la reproducibilidad.
