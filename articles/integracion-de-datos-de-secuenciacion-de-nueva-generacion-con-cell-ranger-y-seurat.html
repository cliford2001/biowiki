Integración de datos de secuenciación de nueva generación con Cell Ranger y Seurat

Introducción

La integración de datos de secuenciación de nueva generación (NGS) es un pilar fundamental en el análisis de datos de RNA de célula única (scRNA-seq). Cuando se estudia la heterogeneidad celular en diferentes muestras, condiciones experimentales o incluso especies, es esencial unificar los datos de múltiples experimentos para identificar poblaciones celulares comunes y diferencias biológicas. En este contexto, Cell Ranger y Seurat se posicionan como herramientas clave: Cell Ranger para la generación de matrices de expresión desde datos raw, y Seurat para el análisis, integración y visualización de los datos.

La integración permite reducir el sesgo técnico (por ejemplo, diferencias en la eficiencia de captura, profundidad de secuenciación o procesamiento) y mejorar la potencia estadística para detectar cambios biológicos reales. Este artículo explora las mejores prácticas y técnicas para integrar datos scRNA-seq utilizando Cell Ranger y Seurat en R, con énfasis en herramientas como Harmony y BBKNN.

Procesamiento con Cell Ranger

Cell Ranger, desarrollado por 10x Genomics, es el pipeline estándar para el análisis de datos scRNA-seq de alta calidad. Su flujo de trabajo incluye:

  • Alineamiento de lecturas a un genoma de referencia (usando STAR o alineador interno).
  • Asignación de genes (feature counting) y generación de una matriz de expresión (barcodes × genes).
  • Control de calidad por medio de métricas como número de genes detectados por célula, porcentaje de reads en genes, y umbral de expresión.

El comando básico para generar la matriz es:

cellranger count \
  --id=sample1 \
  --reference=/path/to/refdata-cellranger-GRCh38-2020-A \
  --fastqs=/path/to/fastq \
  --sample=sample1

Este paso produce un directorio `sample1/outs/filtered_feature_bc_matrix` con la matriz final, que se utiliza como entrada para Seurat.

Integración con Seurat

Una vez que se tienen matrices de expresión de múltiples muestras, Seurat permite unificar los datos mediante métodos de integración. El enfoque más común consiste en:

  1. Crear un objeto Seurat por muestra.
  2. Normalizar los datos (por ejemplo, con NormalizeData()).
  3. Identificar genes variables (FindVariableFeatures()).
  4. Escalar los datos (ScaleData()).
  5. Calcular componentes principales (RunPCA()).
  6. Aplicar un algoritmo de integración: Harmony o BBKNN.

A continuación se muestra un ejemplo de integración con Harmony:

library(Seurat)
library(harmony)

# Cargar datos de cada muestra
sample1 <- Read10X(data.dir = "sample1/outs/filtered_feature_bc_matrix")
sample2 <- Read10X(data.dir = "sample2/outs/filtered_feature_bc_matrix")

# Crear objetos Seurat
seurat1 <- CreateSeuratObject(counts = sample1, project = "sample1")
seurat2 <- CreateSeuratObject(counts = sample2, project = "sample2")

# Normalización y escalamiento
seurat1 <- NormalizeData(seurat1)
seurat1 <- FindVariableFeatures(seurat1, nfeatures = 2000)
seurat1 <- ScaleData(seurat1, features = VariableFeatures(seurat1))
seurat1 <- RunPCA(seurat1, features = VariableFeatures(seurat1))

seurat2 <- NormalizeData(seurat2)
seurat2 <- FindVariableFeatures(seurat2, nfeatures = 2000)
seurat2 <- ScaleData(seurat2, features = VariableFeatures(seurat2))
seurat2 <- RunPCA(seurat2, features = VariableFeatures(seurat2))

# Integración con Harmony
integrated <- IntegrateData(anchorset = anchor_set, features.to.integrate = variable_features)
integrated <- RunPCA(integrated, verbose = FALSE)
integrated <- RunUMAP(integrated, reduction = "pca", dims = 1:30)
integrated <- FindNeighbors(integrated, reduction = "pca", dims = 1:30)
integrated <- FindClusters(integrated, resolution = 0.5)

Alternativamente, BBKNN puede usarse con un enfoque basado en vecinos comunes:

library(BBKNN)

# Unir datos en un listado
seurat_list <- list(seurat1, seurat2)

# Aplicar BBKNN
integrated <- bbknn::bbknn(seurat_list, batch = c("sample1", "sample2"), k = 10)
integrated <- RunPCA(integrated, verbose = FALSE)
integrated <- RunUMAP(integrated, reduction = "pca", dims = 1:30)
integrated <- FindNeighbors(integrated, reduction = "pca", dims = 1:30)
integrated <- FindClusters(integrated, resolution = 0.5)

Resumen

La integración de datos scRNA-seq mediante Cell Ranger y Seurat es un proceso esencial para el análisis de heterogeneidad celular en múltiples condiciones. Cell Ranger proporciona la base de datos de alta calidad, mientras que Seurat, con herramientas como Harmony y BBKNN, permite unificar datos técnicamente distintos, reduciendo sesgos y revelando poblaciones celulares compartidas. La elección entre métodos depende del número de muestras, la profundidad de integración requerida y la distribución de los datos. Un manejo cuidadoso de la normalización, la selección de genes variables y el ajuste de parámetros es crucial para obtener resultados biológicamente relevantes.

Errores comunes / buenas prácticas

  • Usar matrices sin procesar: Siempre normaliza y escala los datos antes de integrar.
  • Ignorar el batch effect: Asegúrate de que el método de integración tenga en cuenta la variable de batch (por ejemplo, muestra, día, laboratorio).
  • Seleccionar demasiados genes variables: Puede introducir ruido. Usar 1000–3000 genes variables suele ser suficiente.
  • Aplicar integración sin validar: Verifica la integración con UMAP y clusters: células de diferentes muestras deben estar mezcladas, no separadas.
  • Usar resolución demasiado alta: Puede fragmentar poblaciones celulares reales. Prueba con resolution = 0.5 como punto de partida.

En resumen, la integración bien ejecutada permite descubrir patrones biológicos ocultos por el ruido técnico. Tener un enfoque sistemático y revisar siempre los resultados visuales es clave para una investigación robusta.
