Integración de datos de expresión con ATAC-seq mediante SCENIC y AUCell

¿Qué es la integración de datos de expresión con ATAC-seq?

La integración de datos de expresión génica (RNA-seq) con datos de accesibilidad cromatina (ATAC-seq) permite una comprensión más profunda de la regulación génica en células individuales. Mientras que el RNA-seq revela qué genes están activos, el ATAC-seq muestra dónde está abierta la cromatina, lo que indica potencial actividad reguladora. Al combinar ambos datos, se puede inferir cómo los factores de transcripción (TFs) regulan la expresión génica en distintos estados celulares.

Por qué es importante

El análisis integrado permite:

  • Identificar reguladores clave en poblaciones celulares específicas.
  • Construir redes de regulación génica (GRNs) a escala single-cell.
  • Explicar la heterogeneidad funcional en tejidos complejos, como el cerebro o el tumor.
  • Descubrir nuevas relaciones entre elementos reguladores y genes diana.

Esta integración es fundamental para avanzar en el estudio del desarrollo, la diferenciación celular y la patología.

Introducción a SCENIC y AUCell

SCENIC (Single-Cell rEgulatory Network Inference and Clustering) es una herramienta robusta para inferir redes de regulación génica a partir de datos de expresión y ATAC-seq. Se basa en la premisa de que los factores de transcripción (TFs) que regulan genes en una célula también tienden a actuar en conjunto, generando patrones coherentes de actividad.

SCENIC utiliza dos pasos principales:

  1. Inferencia de GRNs: A partir de las regiones promotoras (como las detectadas por ATAC-seq), SCENIC identifica los TFs potenciales que podrían estar regulando genes cercanos, usando datos de motivos de TFs (por ejemplo, de JASPAR o HOCOMOCO).
  2. Estimación de actividad de TFs (AUCell): Calcula una puntuación de actividad para cada TF en cada célula, basándose en la expresión de sus genes diana (es decir, genes con motivos en regiones accesibles).

La herramienta AUCell (AUtomated Cell-type specific Expression) se utiliza para calcular estas puntuaciones de actividad. Es especialmente útil porque puede manejar datos de alta dimensionalidad y es robusta frente al ruido biológico.

Flujo de trabajo de integración

El flujo típico sigue estos pasos:

  1. Preprocesamiento: Normalización y filtrado de datos de RNA-seq (con herramientas como Seurat o Scanpy) y ATAC-seq (con CellRanger, ArchR o Signac).
  2. Construcción del GRN: Usar SCENIC para construir una red de regulación, combinando regiones accesibles con motivos TFs.
  3. Cálculo de puntuaciones de actividad: Aplicar AUCell para obtener puntuaciones de actividad de TFs en cada célula.
  4. Visualización y análisis: Clustering, visualización de la actividad de TFs (por ejemplo, con UMAP o TSNE), y correlación con expresión génica.

Este enfoque permite no solo identificar poblaciones celulares, sino también inferir sus programas de regulación internos.

Errores comunes y buenas prácticas

Errores comunes

  • Uso de datos sin preprocesar: No filtrar células de baja calidad o genes con baja expresión puede distorsionar las redes.
  • Ignorar el ruido técnico: No considerar el sesgo de fragmentación en ATAC-seq puede llevar a falsos positivos en la detección de regiones accesibles.
  • Modelos de motivos inadecuados: Usar motivos TFs no validados o no específicos del organismo puede generar redes poco fiables.
  • Interpretación excesiva: Asumir que una alta puntuación de actividad de un TF implica necesariamente su regulación directa.

Buenas prácticas

  • Valida los resultados con datos externos (por ejemplo, ChIP-seq o perturbaciones funcionales).
  • Usa múltiples herramientas de validación (como SCENIC en Python o R).
  • Aplica corrección de múltiples pruebas al analizar la significancia de las puntuaciones.
  • Visualiza siempre las regiones de accesibilidad y sus genes diana asociados.
  • Considera el contexto biológico: un TF puede tener roles diferentes en distintos tejidos.

Resumen

La integración de datos de expresión con ATAC-seq mediante SCENIC y AUCell es una estrategia poderosa para descifrar la regulación génica a nivel single-cell. Al combinar la accesibilidad cromatina con la expresión génica, se pueden inferir redes de regulación y estimar la actividad de factores de transcripción en cada célula, lo que permite explorar la heterogeneidad funcional de tejidos complejos. Aunque el método tiene sus desafíos, como el ruido técnico y la interpretación de los motivos, sigue siendo una herramienta indispensable en el arsenal del bioinformático moderno.
