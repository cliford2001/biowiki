Modelado de interacciones proteína-proteína con STRING y Cytoscape

Introducción: Por qué modelar interacciones proteína-proteína

Las interacciones proteína-proteína (PPI, por sus siglas en inglés) son fundamentales para la función celular, regulación génica, señalización y respuesta a estímulos. Comprender estas redes permite descifrar mecanismos biológicos complejos, identificar dianas terapéuticas en enfermedades como el cáncer o trastornos neurodegenerativos, y validar resultados de experimentos de alto rendimiento como proteómica de afinidad o secuenciación de ARN. En este artículo, exploramos cómo utilizar el recurso público STRING (Search Tool for the Retrieval of Interacting Genes/Proteins) y la herramienta Cytoscape para modelar, visualizar y analizar interacciones proteína-proteína de manera robusta y visualmente informativa.

Construcción de redes de interacciones con STRING

STRING es una base de datos de interacciones proteína-proteína que integra evidencia de múltiples fuentes: interacciones experimentales directas (como co-inmunoprecipitación), evidencia de bases de datos (como BioGRID), predicciones basadas en genómica (como conservación de genes vecinos), y evidencia de coexpresión. Su principal ventaja es que ofrece un puntaje de confianza (confidence score) para cada interacción, que varía entre 0 y 1, permitiendo filtrar redes según la fiabilidad deseada.

Acceso a STRING y entrada de proteínas

Para comenzar, accede a https://string-db.org. Ingresa una lista de proteínas de interés, identificadas por sus nombres de genes, IDs de proteína (como UniProtKB), o secuencias. Puedes ingresar múltiples proteínas separadas por espacios o en una lista. STRING también soporta la carga de archivos.

Personalización de la red

Después de cargar las proteínas, puedes ajustar los parámetros clave:

  • Confianza mínima: Filtra interacciones con un score de confianza inferior al valor establecido (por ejemplo, 0.4 para redes más conservadoras).
  • Mostrar interacciones entre miembros: Si quieres una red densa, deja activa esta opción.
  • Mostrar interacciones con otros genes: Útil para obtener contextos más amplios.
  • Esquema de color: Puedes colorear nodos por funciones, tipos de interacción o niveles de expresión.

Una vez configurado, haz clic en "Buscar". La red se generará y mostrará en el navegador. Puedes exportarla en varios formatos, siendo JSON y TSV los más comunes para su posterior análisis en Cytoscape.

Visualización y análisis en Cytoscape

Cytoscape es una plataforma de software de código abierto diseñada para la visualización, análisis e integración de redes biológicas. Es ideal para profundizar en las redes generadas por STRING, aplicar análisis de topología y personalizar visualizaciones.

Importar redes desde STRING

En Cytoscape, ve a File > Import > Network from File. Elige el archivo TSV o JSON exportado desde STRING. Asegúrate de que el formato sea reconocido correctamente. Cytoscape reconstruirá la red con nodos (proteínas) y aristas (interacciones).

Personalización visual

Con la red cargada, puedes mejorar su legibilidad y valor informativo:

  • Disposición de la red: Usa algoritmos como Force-directed o Organic para organizar los nodos de forma clara.
  • Estilos de nodos: Cambia el tamaño, color y forma según atributos como el grado (número de interacciones), función celular, o expresión diferencial.
  • Etiquetas: Mostrar nombres de proteínas o IDs.
  • Colores de aristas: Asigna colores según el tipo de evidencia (experimento, coexpresión, etc.).

Análisis de topología y funciones biológicas

Cytoscape ofrece múltiples complementos (apps) para análisis avanzado:

  • ClusterMaker2: Identifica comunidades o módulos funcionales dentro de la red usando algoritmos como MCODE o Greedy Modularity.
  • BiNGO: Realiza análisis de enriquecimiento funcional (GO) sobre los nodos en un módulo.
  • NetworkAnalyzer: Calcula métricas como grado, centralidad, coeficiente de clustering, caminos más cortos.

Resumen

El modelado de interacciones proteína-proteína con STRING y Cytoscape combina la potencia de la integración de datos de múltiples fuentes con herramientas visuales y analíticas de alto nivel. STRING permite crear redes basadas en evidencia confiable, mientras que Cytoscape permite transformar esas redes en modelos interpretables, realizar análisis topológicos profundos y visualizar hallazgos de manera efectiva. Esta combinación es esencial en bioinformática funcional, especialmente en el descubrimiento de rutas biológicas, identificación de dianas terapéuticas y comprensión de enfermedades complejas.

Errores comunes / buenas prácticas

  • Ignorar el score de confianza: No filtrar interacciones por confianza puede introducir ruido. Siempre revisa el score mínimo recomendado (por ejemplo, 0.4 para datos experimentales).
  • Redes excesivamente grandes: Cargar muchas proteínas puede generar redes irreconocibles. Comienza con un conjunto de interés y amplía según sea necesario.
  • Confundir tipos de interacción: Las interacciones predichas (por coexpresión) no implican interacción física directa. Usa la visualización en Cytoscape para distinguir tipos de evidencia.
  • Usar solo un algoritmo de disposición: Prueba diferentes algoritmos para encontrar la mejor representación visual.
  • Desactivar análisis funcionales: No dejes de usar BiNGO o ClusterMaker2 para descubrir funciones biológicas subyacentes.
  • Exportar en formato adecuado: Usa TSV para redes simples y JSON para redes con metadatos complejos.
