Análisis de expresión de genes en tejidos con single-cell spatial transcriptomics

¿Qué es el single-cell spatial transcriptomics?

El single-cell spatial transcriptomics (scST) es una tecnología emergente que permite medir la expresión génica a nivel de célula individual, al mismo tiempo que preserva la información espacial sobre la ubicación de cada célula dentro del tejido. A diferencia de las técnicas tradicionales como scRNA-seq, que destruyen la organización tisular durante la disociación celular, scST mantiene el contexto morfológico y arquitectónico del tejido, ofreciendo una visión sin precedentes de cómo las células interactúan en su entorno natural.

Esto es especialmente valioso en campos como la biología del cáncer, el desarrollo embrionario, la inmunología y la neurociencia, donde la localización celular es clave para entender procesos biológicos. Herramientas como 10x Genomics Visium, Slide-seq, Seq-Scope y ST (spatial transcriptomics) de Arraystar permiten la captura de transcritos en puntos espaciales del tejido, generando matrices de expresión que combinan coordenadas espaciales con perfiles génicos.

Flujo de análisis: de los datos brutos al perfil funcional

El análisis de datos scST sigue un flujo estructurado que combina métodos de alineamiento, cuantificación, normalización, reducción de dimensionalidad y análisis espacial. A continuación se detalla cada etapa:

1. Alineamiento de lecturas

Los datos de secuenciación se alinean al genoma de referencia utilizando herramientas como STAR o Salmon. Para scST, es crucial usar un alineador que maneje correctamente los barcodes espaciales, que identifican el punto de captura en el tejido. STAR puede usarse con el parámetro --outSAMtype BAM SortedByCoordinate para generar archivos BAM ordenados, necesarios para la posterior cuantificación.

STAR \
  --genomeDir /path/to/genome \
  --readFilesIn R1.fastq.gz R2.fastq.gz \
  --outFileNamePrefix scst_ \
  --outSAMtype BAM SortedByCoordinate \
  --outSAMattributes All

2. Cuantificación espacial

Una vez alineadas las lecturas, se cuenta el número de transcritos por punto espacial. Herramientas como CellRanger (10x Genomics) o Seurat en R permiten este paso. CellRanger utiliza un algoritmo de asignación de reads basado en los barcodes de captura y los spatial barcodes. En R, se puede usar el paquete Seurat con la función Read10X() para importar datos de Visium.

library(Seurat)

# Importar datos de Visium
visium_data <- Read10X(
  data.dir = "./filtered_feature_bc_matrix/",
  gene.column = 1
)

# Crear objeto Seurat
sobj <- CreateSeuratObject(counts = visium_data, project = "scST_analysis")

3. Preprocesamiento y normalización

El objeto Seurat se preprocesa con NormalizeData() y FindVariableFeatures(). La normalización típica incluye la escala de los datos por el número de genes detectados por célula (total count per cell), y la reducción de efectos de confusión por profundidad de secuenciación.

sobj <- NormalizeData(sobj)
sobj <- FindVariableFeatures(sobj, selection.method = "vst", nfeatures = 2000)

4. Reducción de dimensionalidad y clustering

Se aplica PCA con RunPCA(), seguido de UMAP o TSNE para visualizar la estructura de las células. El clustering se realiza con FindNeighbors() y FindClusters().

sobj <- RunPCA(sobj, features = VariableFeatures(sobj))
sobj <- FindNeighbors(sobj, dims = 1:30)
sobj <- FindClusters(sobj, resolution = 0.5)
sobj <- RunUMAP(sobj, reduction = "pca", dims = 1:30)

5. Análisis espacial

La característica distintiva de scST es el análisis espacial. Se pueden usar funciones de Seurat como SpatialFeaturePlot() para visualizar la expresión de genes clave en el tejido, o FindSpatialModules() para identificar patrones de coexpresión espacial. Además, herramientas como Scanpy en Python permiten análisis avanzados con spatial.py para modelos de vecindad espacial.

library(Seurat)

# Visualizar expresión del gen "ACTB" en el tejido
SpatialFeaturePlot(sobj, features = "ACTB")

Resumen

El análisis de expresión génica en tejidos mediante single-cell spatial transcriptomics representa un avance fundamental en la biología de sistemas. Al combinar la resolución celular con la información espacial, permite descubrir nuevos tipos celulares, entender la arquitectura tisular, identificar zonas de inflamación o invasión tumoral, y revelar interacciones celulares en su contexto natural. Aunque el flujo de análisis es complejo, herramientas como Seurat, Scanpy, CellRanger y STAR ofrecen un ecosistema robusto y bien documentado para llevarlo a cabo. La precisión en el preprocesamiento, la elección adecuada de parámetros de clustering y la interpretación cuidadosa de los patrones espaciales son clave para obtener conclusiones biológicamente significativas.

Errores comunes / buenas prácticas

  • Usar datos sin preprocesamiento: Nunca analizar datos sin normalización ni eliminación de células de baja calidad. Filtra células con low gene count o high mitochondrial content.
  • Ignorar la calidad espacial: Asegúrate de que los spatial barcodes estén correctamente asociados a coordenadas. Revisa la matriz de captura original.
  • Seleccionar parámetros de clustering sin validación: Prueba diferentes valores de resolution en FindClusters() y evalúa los resultados con visualización espacial.
  • No validar con marcadores conocidos: Usa genes de marcadores celulares bien caracterizados (por ejemplo, CD3E para linfocitos, FOXP3 para Treg) para confirmar la identidad celular.
  • Evitar sesgos por profundidad de secuenciación: Usa métodos como log-normalization o scran si hay variación técnica entre puntos.
