Visualización de datos multiómicos con Cicero y scRNA-seq integrado

Introducción: El poder de la visualización en datos multiómicos

La biología moderna se ha convertido en una ciencia de datos, donde el acceso a múltiples capas de información biológica —como expresión génica, accesibilidad cromatina, variación genética y regulación epigenética— permite una comprensión más profunda de los procesos celulares. Sin embargo, el verdadero desafío no reside en la generación de estos datos, sino en su integración y visualización coherente.

Herramientas como Cicero, desarrollada por el laboratorio de Chen et al., han emergido como pilares en la visualización y análisis de datos multiómicos. Cicero no solo integra datos de secuenciación de RNA de célula única (scRNA-seq) con datos de accesibilidad de cromatina (scATAC-seq), sino que también permite la inferencia de redes reguladoras a través de la correlación espacial de promotores y enhancers. Este artículo explora cómo Cicero puede ser utilizado para visualizar de forma integrada la complejidad del genoma en células individuales, destacando su funcionalidad, integración con otros flujos de trabajo y mejores prácticas.

Desarrollo

Cicero: Integración de scRNA-seq y scATAC-seq

Cicero está diseñado para trabajar con matrices de expresión de genes y accesibilidad de cromatina de célula única. Aunque no es un alineador ni un cuantificador, opera sobre datos procesados previamente —por ejemplo, con STAR para alineamiento de RNA o snATAC-seq para cromatina— y utiliza una red neuronal profunda para aprender correlaciones entre promotores y regiones reguladoras activas.

El proceso comienza con la creación de un graph de correlación espacial, donde cada par de loci (regiones genómicas) se evalúa por su co-actividad. Esta correlación se traduce en una matriz de conectividad que refleja potenciales interacciones reguladoras entre genes y elementos distales.

Integración con flujos de trabajo de scRNA-seq

Para integrar Cicero dentro de un flujo de trabajo típico de scRNA-seq, se recomienda el uso de herramientas como Seurat o Scanpy para el preprocesamiento, normalización y reducción de dimensionalidad de los datos de expresión. Una vez obtenidos los marcadores celulares y los clusters, se puede aplicar Cicero a un subconjunto de datos (por ejemplo, una población celular específica) para identificar redes reguladoras diferencialmente activas.

La integración se realiza mediante la superposición de las coordenadas genómicas del scATAC-seq con los genes expresados en el scRNA-seq, utilizando índices de accesibilidad como peak calls de MACS2 o ArchR. Cicero entonces asocia cada pico de accesibilidad con los genes cercanos y sus niveles de expresión, generando un mapa de regulación que puede visualizarse en un espacio de baja dimensionalidad.

Herramientas clave para la visualización

La visualización en Cicero se realiza principalmente mediante gráficos de red y matrices de correlación. A continuación se muestran ejemplos de código reales en R y Python para generar estas visualizaciones:

Código en R con Seurat y Cicero

library(Seurat)
library(cicero)

# Cargar datos de expresión y accesibilidad
seurat_obj <- Read10X(data.dir = "path/to/10x/data")
atac_data <- readRDS("path/to/atac/peaks.rds")

# Crear objeto Cicero
cicero_graph <- build_cicero_graph(
    expression_matrix = seurat_obj@assays$RNA@data,
    chromatin_matrix = atac_data
)

# Obtener correlaciones
correlations <- get_correlation_matrix(cicero_graph)

# Visualizar matriz de correlación
heatmap(correlations, col = colorRampPalette(c("blue", "white", "red"))(100))

Código en Python con Scanpy y Cicero

import scanpy as sc
import numpy as np

# Cargar datos
adata = sc.read_h5ad("path/to/adata.h5ad")
atac_matrix = np.load("path/to/atac/peaks.npy")

# Construir gráfico Cicero
from cicero import build_cicero_graph
graph = build_cicero_graph(
    expression_matrix=adata.X,
    chromatin_matrix=atac_matrix,
    gene_names=adata.var_names,
    peak_names=adata.var_names  # Ajustar según datos reales
)

# Obtener correlaciones
correlations = graph.get_correlation_matrix()

# Visualizar
import matplotlib.pyplot as plt
plt.imshow(correlations, cmap='RdBu_r', aspect='auto')
plt.colorbar()
plt.title("Correlación espacial en Cicero")
plt.show()

Resumen

Cicero representa una herramienta fundamental para la visualización y análisis integrado de datos multiómicos, especialmente cuando se combinan scRNA-seq y scATAC-seq. Su capacidad para inferir interacciones reguladoras mediante correlación espacial permite a los investigadores explorar dinámicas genéticas complejas en células individuales, revelando patrones que no serían evidentes con análisis unimodales.

La integración con flujos de trabajo establecidos como Seurat o Scanpy facilita su uso en un contexto bioinformático amplio, mientras que los bloques de código en R y Python proporcionan una base práctica para su implementación.

Errores comunes / buenas prácticas

  • Usar datos sin normalizar: Asegúrate de normalizar expresión génica y accesibilidad antes de pasarlos a Cicero. Usa métodos como SCTransform en Seurat o scVI en Scanpy.
  • Ignorar la dimensionalidad: Cicero puede ser sensible a ruido técnico. Reduce dimensionalidad previamente (PCA, UMAP) para evitar artefactos.
  • Correlación espacial sin contexto biológico: No interpretes correlaciones como relaciones funcionales directas. Usa anotaciones genómicas (ENCODE, UCSC) para validar resultados.
  • Visualización de grandes matrices: Para matrices de correlación grandes, usa submuestreo o agrupamiento previo para mantener claridad visual.
  • Guardar resultados: Exporta los grafos y matrices generados para su reutilización y reproducibilidad.
