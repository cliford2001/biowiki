Visualización de datos de expresión en múltiples tejidos con Single-Cell ATAC-seq y Cicero

Introducción

La biología del desarrollo y la patología requieren comprender cómo se regulan los genes en contextos de tejidos específicos. La secuenciación de ATAC en células individuales (scATAC-seq) permite mapear la accesibilidad cromatina en millones de células individuales, revelando la regulación genómica a nivel de promotores, enhancers y otras regiones reguladoras. Sin embargo, la interpretación de estos datos es compleja, especialmente cuando se compara la regulación entre múltiples tejidos. En este artículo, exploramos cómo combinar scATAC-seq con el método Cicero para visualizar y analizar patrones de expresión regulada en múltiples tejidos, proporcionando una ventana hacia la arquitectura de red genética subyacente.

Desarrollo

Primeros pasos: Preprocesamiento de datos scATAC-seq

Antes de cualquier análisis de regulación, es esencial preprocesar los datos scATAC-seq. Esto incluye la alineación de reads, eliminación de contaminantes mitocondriales y de células basura, y la generación de matrices de accesibilidad. Herramientas como cellranger-atac (10x Genomics) o scATAC-seq pipeline basado en bowtie2 y macs2 son ampliamente utilizadas.

cellranger-atac count --id=atac_pipeline \
  --fastqs=/path/to/fastqs \
  --reference=/path/to/reference \
  --sample=sample1 \
  --output-dir=./output

Construcción de redes reguladoras con Cicero

Cicero, desarrollado por los laboratorios de Chen et al., utiliza la correlación de accesibilidad entre regiones del genoma para predecir relaciones de regulación genómica entre enhancers y promotores. Al aplicar Cicero a múltiples tejidos, es posible identificar redes reguladoras específicas de tejido y sus diferencias.

library(Cicero)

# Cargar matriz de accesibilidad
atac_data <- readRDS("./output/atac_matrix.rds")

# Construir red de regulación
cicero_data <- build_cicero_graph(atac_data, 
                                  genome = "hg38", 
                                  min_correlation = 0.6)

# Guardar resultados
saveRDS(cicero_data, "./output/cicero_network.rds")

Visualización en múltiples tejidos

Para comparar redes entre tejidos, es fundamental normalizar y visualizar las predicciones de Cicero de forma integrada. ggplot2 y igraph en R permiten crear mapas de redes ajustables que destacan diferencias de conectividad.

library(ggplot2)
library(igraph)

# Cargar datos de múltiples tejidos
tissue1 <- readRDS("./output/cicero_tissue1.rds")
				tissue2 <- readRDS("./output/cicero_tissue2.rds")

# Crear gráficos separados
p1 <- plot_cicero_network(tissue1, 
                          title = "Tejido 1: Hígado")
p2 <- plot_cicero_network(tissue2, 
                          title = "Tejido 2: Riñón")

# Mostrar en paralelo
library(gridExtra)
ggarrange(p1, p2, ncol = 2)

Análisis de diferencias entre tejidos

Construir matrices de diferencias de conectividad permite identificar hub genes o regiones reguladoras clave que varían entre tejidos. Estas diferencias pueden asociarse con funciones biológicas específicas.

# Calcular diferencias en conectividad
conn_diff <- compute_connectivity_difference(tissue1, tissue2)

# Encontrar top 10 regiones más diferenciadas
top_diff <- head(sort(conn_diff, decreasing = TRUE), 10)

# Visualizar
barplot(top_diff, main = "Top 10 regiones diferenciadas entre tejidos")

Integración con datos de expresión genética

Para vincular la accesibilidad cromatina con la expresión génica, se puede integrar Cicero con scRNA-seq. Herramientas como Seurat o Scanpy permiten alinear datos de ambos tipos y correlacionar las predicciones de Cicero con la expresión real.

# En Seurat
library(Seurat)

# Cargar datos scRNA-seq
rna_data <- Read10X("./rna_data/" )

# Intercalar con datos de cicero
joint_data <- IntegrateData(
  object1 = rna_data,
  object2 = cicero_data,
  dims = 1:30,
  verbose = FALSE
)

# Visualizar en UMAP
DimPlot(joint_data, group.by = "tissue")

Resumen

La combinación de scATAC-seq y Cicero ofrece una potente estrategia para visualizar y analizar la regulación genómica en múltiples tejidos. Mediante el análisis de redes de conectividad y su integración con datos de expresión, es posible descubrir circuitos genéticos específicos de tejido, identificar genes reguladores clave y comprender cómo la arquitectura cromatina varía entre condiciones biológicas. Esta aproximación es esencial para la investigación en biología del desarrollo, enfermedades y terapias génicas.

Errores comunes / buenas prácticas

  • Evitar correlaciones espurias: Fijar un umbral mínimo de correlación (por ejemplo, 0.6) en Cicero para evitar relaciones artificiales.
  • Verificar calidad de datos: Usar cellranger-atac o scATAC-seq QC para evaluar la calidad de los datos antes de construir redes.
  • Normalizar entre tejidos: Asegurarse de que las matrices de accesibilidad sean comparables mediante normalización (por ejemplo, con scater o Seurat).
  • Validar predicciones: Utilizar datos de ChIP-seq o CRISPRi para validar las predicciones de regulación de Cicero.
  • Visualizar con cuidado: Evitar superposiciones densas en gráficos de redes; usar herramientas como ggraph para mejor presentación.
