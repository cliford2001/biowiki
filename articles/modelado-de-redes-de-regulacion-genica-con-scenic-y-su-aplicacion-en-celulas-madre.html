Modelado de redes de regulación génica con SCENIC y su aplicación en células madre

Introducción: El poder de las redes reguladoras en biología celular

La regulación génica es el proceso fundamental que determina qué genes se activan o desactivan en cada célula, en cada momento, y bajo cada condición. Aunque los datos de expresión génica (como los obtenidos mediante RNA-seq) nos muestran qué genes están activos, no nos revelan directamente quiénes son los reguladores que controlan esta actividad. Este es precisamente el desafío al que se enfrenta el análisis de scRNA-seq (secuenciación de ARN a nivel de célula única).

El modelo SCENIC (Single-Cell rEgulatory Network Inference and Clustering) surge como una herramienta poderosa para abordar esta limitación. SCENIC permite reconstruir redes de regulación génica a partir de datos de expresión de células únicas, identificando no solo qué genes se expresan, sino también qué factores de transcripción (TFs) podrían estar dirigiendo esa expresión. Su aplicación en el estudio de células madre es particularmente valiosa, ya que estas células tienen un estado de regulación altamente dinámico, y entender sus redes reguladoras es clave para desentrañar los mecanismos que controlan su pluripotencia, diferenciación y reprogramación.

¿Cómo funciona SCENIC? Un enfoque basado en regulación y expresión

SCENIC opera en dos etapas principales: la inferencia de reguladores y la construcción de redes reguladoras.

1. Identificación de reguladores activos (Regulon Inference)

SCENIC comienza por identificar qué factores de transcripción (TFs) están activos en cada célula. Para ello, utiliza una base de datos de motivos de unión a ADN (como CISBP-RNA o Roadmap Epigenomics) y analiza si los genes que se expresan en una célula tienen promotores con estos motivos. Si un TF tiene muchos genes objetivo con expresión alta, se considera que ese TF está activo en esa célula.

Este proceso genera lo que se conoce como un regulón (regulon): un conjunto de genes controlados por un TF específico. Cada célula tiene un perfil de regulones activos, que refleja su estado regulador.

2. Construcción de la red reguladora

Posteriormente, SCENIC utiliza el perfil de regulones activos en cada célula para realizar clustering (por ejemplo, con Leiden o louvain), lo que permite identificar grupos de células con estados reguladores similares. Esto es crucial para el análisis de células madre, donde pequeñas variaciones en la actividad de reguladores pueden marcar diferenciaciones incipientes.

Finalmente, se puede aplicar SCENIC+ (SCENIC+), una extensión que combina la información de regulones con datos de epigenética (como ChIP-seq o ATAC-seq) para aumentar la precisión en la inferencia de redes.

Aplicación en células madre: Descifrando la pluripotencia y la diferenciación

Las células madre embrionarias (mESCs) y las células madre inducidas (iPSCs) representan un sistema ideal para aplicar SCENIC. Su estado de pluripotencia está mantenido por una red de reguladores clave, como Oct4, Sox2, Klf4, y c-Myc (los factores de Yamanaka).

Al aplicar SCENIC a un conjunto de células madre, se pueden identificar:

  • Las células con un estado de pluripotencia robusto (alta actividad de Oct4/Sox2).
  • Las células en proceso de transición hacia la diferenciación (baja actividad de pluripotencia y aumento de reguladores de linajes específicos).
  • Subpoblaciones de células madre con distintas “fases” de reprogramación o estabilidad.

Esto permite a los investigadores no solo clasificar las células, sino también proponer hipótesis sobre los factores críticos que desencadenan la diferenciación. Por ejemplo, si se observa que un regulón de Sox17 se activa en un subconjunto de células, se puede inferir que es un candidato clave en la ruta de diferenciación hacia endodermo.

Bloques de código clave en R y Python

SCENIC está disponible principalmente como una suite de herramientas en R (paquete SCENIC) y Python (scanpy con el paquete scenic).

En R (usando SCENIC)

library(SCENIC)

# Cargar datos de expresión (matrix de genes x células)
data <- readRDS("expression.rds")

# Generar la matriz de regulones
regulon_matrix <- createRegulonMatrix(data, 
                                      motifs = "motifs.cisbp.rna", 
                                      genome = "mm10")

# Calcular el score de regulón para cada célula
regulon_scores <- calculateRegulonScores(regulon_matrix)

# Clusterizar células basado en los scores
clusters <- clusterCells(regulon_scores)

# Visualizar resultados
plotRegulonScores(regulon_scores, clusters)

En Python (usando scanpy + scenic)

import scanpy as sc
import pandas as pd

# Cargar datos
adata = sc.read_h5ad("expression.h5ad")

# Preprocesamiento (normalización, logaritmo, selección de genes)
sc.pp.normalize_total(adata, target_sum=1e4)
sc.pp.log1p(adata)
sc.pp.highly_variable_genes(adata, n_top_genes=2000)

# Construir regulones
sc.tl.rank_genes_groups(adata, groupby="clusters", method="t-test")

# Aplicar SCENIC
scenic = sc.tl.scenic(adata, 
                      motif_database="cisbp_rna", 
                      genome="mm10")

# Clustering de células basado en regulones
sc.tl.leiden(adata, resolution=0.8)

# Visualizar
sc.pl.umap(adata, color=["leiden", "regulon_score_Oct4"], legend_loc="on data")

Resumen

SCENIC transforma el análisis de datos de células únicas al permitir la inferencia de redes de regulación génica, pasando de un enfoque puramente genómico (qué genes están expresados) a uno funcional (qué factores de transcripción los están regulando). Su aplicación en células madre es particularmente potente, ya que permite mapear los estados de pluripotencia y los primeros pasos de diferenciación con una resolución sin precedentes. Al integrar datos de expresión con información de motivos de unión, SCENIC ofrece una ventana única hacia los circuitos reguladores que definen el destino celular.

Errores comunes / buenas prácticas

  • Usar una base de datos de motivos inadecuada: Asegúrese de elegir una base de datos que sea relevante para el organismo y el tipo celular (por ejemplo, cisbp_rna para humanos/murinos).
  • Ignorar la calidad de los datos: SCENIC requiere datos de expresión bien procesados. No aplique SCENIC directamente a raw counts sin normalización.
  • Asumir causalidad: SCENIC infiere correlaciones reguladoras, no causalidad. Un alto score de regulón no implica que el TF sea el causante directo.
  • Suponer que todos los regulones son biológicamente relevantes: Filtrar por puntuación y número de genes objetivo para enfocarse en regulones fuertes.
  • Olvidar validar con datos independientes: Si se tiene acceso a datos de ChIP-seq o ATAC-seq, utilícelos para validar las predicciones de SCENIC.
