Integración de datos genómicos con Multi-Omics Factor Analysis

¿Qué es la integración de datos genómicos con MOFA?

La análisis de factores multi-ómicos (Multi-Omics Factor Analysis, MOFA) es una técnica estadística avanzada diseñada para integrar múltiples tipos de datos biológicos —como expresión génica, metilación, proteómica, y datos de citometría de flujo— en un único marco analítico. A diferencia de métodos tradicionales que analizan cada conjunto de datos por separado, MOFA busca factores latentes que explican la variabilidad conjunta entre múltiples matrices de datos. Esto permite identificar patrones biológicos subyacentes comunes a varios niveles moleculares, revelando relaciones complejas que de otro modo permanecerían ocultas.

MOFA es especialmente útil en estudios de enfermedades complejas, como el cáncer, donde la interacción entre distintos niveles de regulación (genómica, epigenómica, transcriptómica) juega un papel crucial. Su enfoque permite descubrir subtipos de pacientes, biomarcadores de respuesta a tratamientos, y mecanismos biológicos compartidos que no se detectan mediante análisis univariados.

Principios de MOFA y MOFA2

MOFA se basa en un modelo de factorización de matrices bajo un marco bayesiano. Dado un conjunto de matrices de datos $ X_1, X_2, ..., X_K $, cada una representando un tipo de omica (por ejemplo, RNA-seq, DNA methylation), MOFA modela cada matriz como una combinación lineal de factores comunes y factores específicos de cada omica:

X_k = W_k F + E_k

Donde:

  • W_k: matriz de cargas para la omica k
  • F: matriz de factores latentes (comunes a todas las omicas)
  • E_k: ruido específico de la omica k

Esta formulación permite que MOFA descubra factores que capturan la variabilidad común entre múltiples omicas, así como factores específicos que explican la variabilidad intrínseca de cada tipo de datos.

La versión MOFA2 (implementada en Python) mejora la eficiencia computacional, ofrece mejor soporte para datos de alta dimensión, y cuenta con una interfaz más accesible mediante Scanpy y anndata. Es la herramienta recomendada para estudios modernos debido a su rapidez y flexibilidad.

Implementación con MOFA2 en Python

El siguiente ejemplo muestra cómo usar MOFA2 para integrar datos de expresión génica y metilación usando Scanpy:

import scanpy as sc
import mofa2

# Cargar datos en anndata
adata_gene = sc.read_h5ad('data/expr.h5ad')
adata_meth = sc.read_h5ad('data/meth.h5ad')

# Crear objeto MOFA
mofa = mofa2.MOFA()

# Añadir matrices
mofa.add_data(adata_gene, 'gene_expr')
mofa.add_data(adata_meth, 'methylation')

# Ejecutar análisis
mofa.run(n_factors=10, max_iter=1000)

# Obtener factores
factors = mofa.get_factors()

# Visualizar
mofa.plot_factors()  # Gráficos de factores y cargas
mofa.plot_heatmap()  # Mapa de calor de cargas

Integración con Seurat (R)

En R, el paquete MOFA ofrece una implementación robusta. Aquí se muestra un flujo típico para integrar datos de RNA-seq y ATAC-seq:

library(MOFA)

# Cargar datos
expr <- readRDS("data/expr.rds")
atac <- readRDS("data/atac.rds")

# Crear objeto MOFA
mofa_obj <- create_mofa_object(
  data = list(expr = expr, atac = atac),
  type = "matrix"
)

# Entrenar el modelo
mofa_obj <- run(mofa_obj, n_factors = 8)

# Obtener factores
factors <- get_factors(mofa_obj)

# Visualización
plot(mofa_obj, what = "factors")
plot(mofa_obj, what = "loadings")

Resumen y buenas prácticas

MOFA y MOFA2 son herramientas poderosas para la integración multi-ómica, capaces de descubrir factores biológicos compartidos entre múltiples niveles de regulación. Su aplicación permite un análisis más profundo de la biología celular y patológica, facilitando el descubrimiento de biomarcadores y subtipos de pacientes.

Buenas prácticas:

  • Normalizar cada tipo de dato antes de integrar (por ejemplo, usar DESeq2 para RNA-seq o minfi para metilación).
  • Seleccionar el número de factores con validación cruzada o métodos de selección como el elbow method o variance explained.
  • Verificar la calidad del modelo mediante plot(mofa_obj, "reconstruction") para asegurar que los datos se reconstruyen adecuadamente.
  • Usar scanpy o Seurat para visualizar resultados en espacios de factores y relacionarlos con etiquetas biológicas (clases, estados celulares).
