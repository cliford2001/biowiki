salmon

Repositorio · 914 estrellas · Rust

¿Qué es y para qué sirve?

salmon es una herramienta de alto rendimiento y alta precisión para la cuantificación a nivel de transcripción a partir de lecturas de secuenciación de ARN (RNA-seq). Utiliza alineamiento selectivo (selective alignment) para calcular eficientemente las abundancias de transcripciones, minimizando el costo computacional sin sacrificar precisión. Es especialmente útil en estudios de expresión génica diferencial y en análisis de variantes transcriptómicas.

Instalación

salmon está escrito en Rust y se distribuye como binario precompilado. Puedes instalarlo usando conda o homebrew: 


# Usando conda (recomendado)
conda install -c bioconda salmon

# Usando homebrew (macOS)
brew install salmon

Uso básico

El uso mínimo de salmon implica el índice de transcripciones y la cuantificación. Aquí un ejemplo ilustrativo de un flujo típico con datos simulados: 


# 1. Construir índice de transcripciones
salmon index -t transcripts.fasta -i salmon_index

# 2. Cuantificar lecturas
salmon quant -i salmon_index \
               -r reads.fastq \
               -o quant_output \
               --libType ISF

Sintaxis ilustrativa; verifica en la documentación oficial.

Flujo de trabajo típico

Un flujo típico de análisis con salmon incluye:

  1. Obtener la transcripción de referencia (FASTA).
  2. Construir un índice con salmon index.
  3. Alinear y cuantificar lecturas de RNA-seq con salmon quant.
  4. Exportar resultados para análisis diferencial (ej. con tximport en R).

Errores comunes y buenas prácticas

  • Usar --libType incorrecto: Asegúrate de especificar el tipo de biblioteca correcto (ISF, SF, FR, etc.) según tu protocolo de preparación de biblioteca.
  • No validar el índice: Comprueba que el índice se ha generado sin errores usando salmon index --check.
  • Ignorar el preprocesamiento: Aunque salmon no requiere alineamiento completo, el preprocesamiento de lecturas (cortar adaptadores, filtro de baja calidad) es clave para resultados precisos.
  • Usar tximport en R: Para análisis diferencial, exporta resultados a tximport para obtener matrices de expresión.
