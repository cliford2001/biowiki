BWA

Ver repositorio · 1758 estrellas · C

¿Qué es y para qué sirve?

BWA (Burrow-Wheeler Aligner) es una herramienta de alineamiento de lecturas cortas de secuencias de ADN o ARN contra un genoma de referencia. Está diseñada para alinear secuencias de alto rendimiento (short reads), típicamente generadas por plataformas de secuenciación de próxima generación como Illumina. BWA es ampliamente utilizado en proyectos de biología molecular, genómica y bioinformática clínica para identificar variaciones genéticas, análisis de expresión, y estudios de metagenómica.

Instalación

BWA está escrito en C y se compila desde el código fuente. El método recomendado es clonar el repositorio y compilarlo con make:

git clone https://github.com/lh3/bwa.git
cd bwa
make

Alternativamente, si tienes conda instalado, puedes instalarlo directamente desde el canal bioconda:

conda install -c bioconda bwa

Uso básico

Una vez compilado, BWA puede usarse en varias formas. El flujo más común implica dos pasos: primero, construir un índice del genoma de referencia; segundo, alinear las lecturas a ese índice. Aquí tienes un ejemplo ilustrativo:

# Paso 1: Construir índice del genoma de referencia
bwa index hg38.fa

# Paso 2: Alinear lecturas (pair-ended) a la referencia
bwa mem hg38.fa reads_1.fastq reads_2.fastq > alineamiento.sam

Nota: La sintaxis exacta de los comandos puede variar dependiendo del tipo de entrada (pair-ended, single-ended) y del formato del genoma. Asegúrate de consultar la documentación oficial para el uso exacto según tu caso.

Flujo de trabajo típico

El flujo típico de análisis con BWA sigue estos pasos:

  1. Obtener el genoma de referencia (por ejemplo, hg38.fa).
  2. Crear un índice de BWA con bwa index.
  3. Alinear las lecturas de secuenciación usando bwa mem.
  4. Convertir el archivo SAM a BAM (formato binario) con samtools view -bS alineamiento.sam -o alineamiento.bam.
  5. Ordenar el archivo BAM con samtools sort.
  6. Índice del BAM final para su uso posterior.

Errores comunes y buenas prácticas

  • Índice no generado: Asegúrate de ejecutar bwa index antes de usar bwa mem. Si no hay índice, BWA fallará con un error claro.
  • Formato incorrecto de entrada: BWA requiere que las lecturas estén en formato FASTQ. Si usas otro formato, conviértelo primero con herramientas como fastqc o seqtk.
  • Genoma de referencia no compatible: Usa un genoma de referencia que coincida con el conjunto de datos (por ejemplo, hg38 para humano, mm10 para ratón).
  • No usar bwa mem para lecturas largas: BWA está optimizado para lecturas cortas. Para lecturas largas (como las de Nanopore o PacBio), utiliza minimap2 en su lugar.
  • Práctica recomendada: Siempre verifica la calidad de las lecturas antes del alineamiento con fastqc y corrige posibles contaminantes o secuencias de adaptadores con trimmomatic o cutadapt.
