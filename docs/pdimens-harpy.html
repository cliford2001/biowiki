harpy

Repositorio · 21 estrellas · Python

¿Qué es y para qué sirve?

Harpy es una herramienta de bioinformática desarrollada en Python diseñada para procesar datos de lecturas vinculadas (linked-read data), desde secuencias crudas hasta la fase de haplotipos. También es compatible con datos de secuenciación de genoma completo (WGS). Su objetivo es facilitar el análisis de estructura genómica a gran escala, especialmente en el contexto de la resolución de variantes complejas y la reconstrucción de haplotipos a partir de datos de alto rendimiento.

Instalación

Harpy se distribuye como paquete Python. Puedes instalarlo usando pip:

pip install harpy

Uso básico

Este es un ejemplo ilustrativo de cómo se podría utilizar harpy en un script de Python. La sintaxis real puede variar según la versión y los requisitos específicos del flujo de datos.

import harpy

# Cargar datos de lecturas vinculadas
data = harpy.load_linked_reads("input_linked_reads.fastq")

# Procesar datos y generar haplotipos
phased_data = harpy.process(data, reference="hg38")

# Guardar resultados
phased_data.save("output_haplotypes.vcf")

Flujo de trabajo típico

El flujo habitual con harpy incluye los siguientes pasos:

  1. Carga de datos crudos de secuenciación (FASTQ o similar).
  2. Preprocesamiento y filtrado de lecturas.
  3. Ensamblaje de regiones basadas en datos de lecturas vinculadas.
  4. Reconstrucción de haplotipos (phase inference).
  5. Generación de archivos de salida en formato VCF o BAM.

Errores comunes y buenas prácticas

  • Uso de versiones desactualizadas: Asegúrate de mantener harpy actualizado para aprovechar mejoras de rendimiento y correcciones de errores.
  • Formato de entrada incorrecto: Verifica que los archivos de entrada (FASTQ, BAM, etc.) estén en el formato esperado por harpy. Usa herramientas como fastqc o samtools para validar los datos antes del análisis.
  • Requisitos de memoria: El procesamiento de datos de lecturas vinculadas puede ser intensivo en memoria. Asegúrate de contar con suficiente RAM y considera ejecutar el análisis en un entorno con recursos adecuados.
  • Validación de resultados: Siempre verifica los resultados finales con herramientas de validación como bcftools o vcfcheck para asegurar la integridad de los archivos VCF generados.
