Seurat

Repositorio en GitHub · 2768 estrellas · R

¿Qué es y para qué sirve?

Seurat es una herramienta de análisis de datos de secuenciación de células individuales (scRNA-seq) desarrollada en R. Está diseñada para integrar, visualizar, analizar y explorar datos de expresión génica a nivel de célula individual. Permite identificar poblaciones celulares, detectar diferencias de expresión entre condiciones, realizar análisis de estados celulares y visualizar resultados de forma interactiva.

Instalación

Seurat se instala en R usando el gestor de paquetes de Bioconductor. Asegúrate de tener R y Bioconductor instalados.

if (!require("BiocManager", quietly = TRUE))
  install.packages("BiocManager")

BiocManager::install("Seurat")

Uso básico

Este ejemplo muestra cómo cargar datos, crear un objeto Seurat, realizar filtrado y visualización básica.

library(Seurat)

# Cargar datos de expresión (ejemplo: matriz de genes x células)
data <- Read10X(data.dir = "path/to/your/data")

# Crear objeto Seurat
seurat_obj <- CreateSeuratObject(counts = data)

# Filtrar células por número de genes expresados y por expresión total
seurat_obj <- subset(seurat_obj, subset = nFeature_RNA > 200 & nFeature_RNA < 2500)
seurat_obj <- subset(seurat_obj, subset = nCount_RNA < 10000)

# Normalizar datos
seurat_obj <- NormalizeData(seurat_obj)

# Encontrar genes variables
seurat_obj <- FindVariableFeatures(seurat_obj, selection.method = "vst", nfeatures = 2000)

# Escalar datos
seurat_obj <- ScaleData(seurat_obj)

# Reducción de dimensionalidad
seurat_obj <- RunPCA(seurat_obj, verbose = FALSE)

# Clustering y visualización
seurat_obj <- FindNeighbors(seurat_obj, dims = 1:30)
seurat_obj <- FindClusters(seurat_obj, resolution = 0.5)
seurat_obj <- RunUMAP(seurat_obj, dims = 1:30)

# Visualizar resultados
DimPlot(seurat_obj, group.by = "seurat_clusters")

Flujo de trabajo típico

El flujo estándar de análisis con Seurat incluye:

  1. Carga de datos desde archivos 10x (o formatos compatibles).
  2. Creación del objeto Seurat.
  3. Filtrado de células y genes.
  4. Normalización y escalado.
  5. Identificación de genes variables.
  6. Reducción de dimensionalidad (PCA).
  7. Clustering y visualización (UMAP, t-SNE).
  8. Identificación de marcadores diferenciales y análisis funcional.

Errores comunes y buenas prácticas

  • Problemas de memoria: Seurat puede consumir mucha RAM con grandes conjuntos de datos. Usa el parámetro features en Read10X para cargar solo los genes necesarios.
  • Filtrado inadecuado: Filtra células por número de genes y expresión total para eliminar células basura. Ajusta los umbrales según tus datos.
  • Resolución de clustering: El parámetro resolution en FindClusters controla el número de clústeres. Valores bajos (<0.5) dan menos clústeres; altos (>1.0) pueden sobrecategorizar.
  • Normalización: Asegúrate de normalizar después del filtrado. La normalización no debe hacerse antes de filtrar.
  • Visualización: Usa DimPlot y FeaturePlot para explorar datos antes de tomar decisiones biológicas.
  • Actualizaciones: Seurat se actualiza frecuentemente. Usa BiocManager::install("Seurat") para obtener la última versión.
