cutadapt

https://github.com/marcelm/cutadapt · 586 estrellas · Python

¿Qué es y para qué sirve?

cutadapt es una herramienta de línea de comandos diseñada para eliminar secuencias de adaptadores de lecturas de secuenciación de ADN/RNA. Es especialmente útil en proyectos de secuenciación de nueva generación (NGS), donde las secuencias de adaptadores pueden interferir con el análisis posterior. Esta herramienta identifica y trunca automáticamente las secuencias de adaptadores en los extremos de las lecturas, mejorando así la calidad y precisión del análisis bioinformático.

Instalación

cutadapt está escrito en Python y se puede instalar fácilmente utilizando pip:

pip install cutadapt

Uso básico

Un uso básico de cutadapt implica especificar la secuencia del adaptador que se desea eliminar. Por ejemplo, si se tiene una lectura que comienza con un adaptador conocido, el comando podría ser:

cutadapt -a ADAPTADOR=AGATCGGAAGAGCACACGTCTGAACTCCAGTCAC -o salida.fastq entrada.fastq

Este comando elimina el adaptador AGATCGGAAGAGCACACGTCTGAACTCCAGTCAC de la cabeza de cada lectura en el archivo entrada.fastq, guardando el resultado en salida.fastq. La sintaxis ilustrativa; verifica en la documentación oficial.

Flujo de trabajo típico

El flujo habitual de trabajo con cutadapt es:

  1. Preparar el archivo de lecturas de entrada (fastq o fasta).
  2. Identificar la secuencia del adaptador utilizado durante la librería de secuenciación.
  3. Ejecutar cutadapt para eliminar los adaptadores de las lecturas.
  4. Verificar la salida y proceder con el análisis posterior (alineamiento, ensamblaje, etc.).

Errores comunes y buenas prácticas

  • Adaptador mal especificado: Asegúrate de conocer exactamente la secuencia del adaptador utilizado en tu experimento. Un error en la secuencia puede hacer que cutadapt no detecte los adaptadores, o que elimine secuencias incorrectas.
  • Uso de adaptadores múltiples: Si tu librería incluye adaptadores en ambos extremos, utiliza los parámetros -a y -g para especificar el adaptador 5' y 3' respectivamente.
  • Revisar el informe de salida: cutadapt genera un archivo de informe con estadísticas sobre las lecturas procesadas. Revisa este informe para verificar que la eliminación de adaptadores fue exitosa.
  • Evitar el uso excesivo de parámetros: No apliques filtros innecesarios en la línea de comandos a menos que sean necesarios. El uso de opciones como --minimum-length o --max-n debe ser justificado por el contexto del análisis.
