samtools/samtools

https://github.com/samtools/samtools · 1924 estrellas · C

¿Qué es y para qué sirve?

samtools/samtools es una colección de herramientas escritas en C utilizando la biblioteca htslib, diseñadas para manipular datos de secuenciación de nueva generación (NGS). Permite la lectura, escritura, análisis y transformación de archivos en formato SAM, BAM y CRAM, que son estándares ampliamente utilizados en bioinformática para almacenar datos de alineamiento de lecturas de ADN/RNA.

Instalación

Samtools se puede instalar fácilmente desde el sistema operativo mediante gestores de paquetes. Aquí tienes los comandos reales para las principales distribuciones:

sudo apt-get update
sudo apt-get install samtools

Alternativamente, si se desea la última versión desde el código fuente:

git clone https://github.com/samtools/samtools.git
cd samtools
make
sudo make install

Uso básico

Un uso común es convertir un archivo BAM a su formato texto SAM para inspección, o realizar un resumen del archivo. Aquí tienes un ejemplo realista:

samtools view -h sample.bam | head -10
samtools stats sample.bam | grep "^SN"

Flujo de trabajo típico

En un flujo típico de análisis de NGS, samtools se utiliza para:

  1. Convertir archivos de alineamiento de formato SAM a BAM (compacto y indexado).
  2. Indexar archivos BAM para acceso rápido.
  3. Extraer regiones específicas del genoma.
  4. Contar cobertura de lecturas.
  5. Generar estadísticas de calidad.

Errores comunes y buenas prácticas

  • Olvidar indexar el archivo BAM: muchas operaciones (como samtools view en regiones específicas) requieren un archivo .bai indexado. Siempre ejecuta samtools index sample.bam después de crear el archivo BAM.
  • Uso de formato incorrecto: asegúrate de usar -h con samtools view para mantener los encabezados si necesitas visualizar el archivo SAM completo.
  • Comandos ineficientes: para grandes archivos, considera usar samtools view -b para mantener el formato binario (BAM) en lugar de convertir a texto.
  • Verificar la sintaxis: aunque los comandos mostrados aquí son estándar, siempre consulta la documentación oficial si necesitas flags avanzados.
