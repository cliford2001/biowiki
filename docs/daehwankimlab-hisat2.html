HISAT2

Repositorio en GitHub · 540 estrellas · C++

¿Qué es y para qué sirve?

HISAT2 es una herramienta de alineamiento de secuencias de RNA de última generación que utiliza un índice jerárquico basado en grafos (Graph-based alignment, Hierarchical Graph FM index) para alinear lecturas de secuenciación de nueva generación (NGS) a un genoma de referencia con alta precisión y eficiencia. Está diseñado específicamente para alineamientos de RNA-Seq, incluyendo el manejo de intrones y exones, y es altamente eficaz en el alineamiento de lecturas largas y complejas.

Instalación

HISAT2 está disponible como binario precompilado para múltiples sistemas operativos. Puedes instalarlo usando conda o descargar el binario directamente. 

conda install -c bioconda hisat2

Alternativamente, puedes descargar el binario desde el repositorio oficial: 

curl -L -O https://github.com/DaehwanKimLab/hisat2/releases/latest/download/hisat2-2.2.1-Linux-x86_64.zip
unzip hisat2-2.2.1-Linux-x86_64.zip
sudo mv hisat2-2.2.1/hisat2 /usr/local/bin/

Uso básico

Un ejemplo típico de uso de HISAT2 para alinear lecturas de RNA-Seq: 

hisat2 -x hisat2_index -1 reads_1.fq -2 reads_2.fq -S aligned.sam

Donde:

  • -x: ruta al índice de referencia (generado con hisat2-build).
  • -1 y -2: archivos de lecturas pareadas (fwd y rev).
  • -S: salida en formato SAM.

Flujo de trabajo típico

El flujo típico para analizar datos de RNA-Seq con HISAT2 incluye:

  1. Construir el índice de referencia con hisat2-build.
  2. Alinear las lecturas a ese índice usando hisat2.
  3. Convertir el archivo SAM a BAM (si es necesario) con samtools view.
  4. Ordenar y indexar el BAM con samtools sort y samtools index.

Errores comunes y buenas prácticas

Errores comunes:

  • Usar un índice de referencia incorrecto o incompleto (asegúrate de haber ejecutado hisat2-build correctamente).
  • Olvidar especificar -1 y -2 para lecturas pareadas.
  • Probar con archivos de entrada en formato incorrecto (por ejemplo, FASTA en lugar de FASTQ).

Buenas prácticas:

  • Siempre usa hisat2-build para generar el índice desde el genoma de referencia (FASTA).
  • Considera usar el parámetro -p para paralelizar el alineamiento en múltiples núcleos.
  • Para datos de RNA-Seq, asegúrate de que el genoma incluya información de splicing (por ejemplo, mediante archivos GTF).
  • Usa samtools sort y samtools index para preparar el archivo BAM final.
