SigProfilerTopography

Repositorio · Autores: 24 estrellas · Python

¿Qué es y para qué sirve?

SigProfilerTopography es una herramienta de la suite SigProfilerSuite que permite evaluar el efecto de la organización cromatínica, modificaciones de histonas, unión de factores de transcripción, replicación del ADN y transcripción sobre la actividad de diferentes procesos mutacionales. Esta herramienta revela las características topográficas únicas de las firmas mutacionales, ayudando a comprender cómo los factores de la estructura y dinámica del genoma influyen en la aparición de mutaciones.

Instalación

pip install SigProfilerTopography

Uso básico

import SigProfilerTopography as topography

# Cargar una firma mutacional
topography.run_topography(
    input_file="mutational_signatures.csv",
    output_directory="output/",
    reference_genome="hg38"
)

Flujo de trabajo típico

El flujo habitual consiste en: 1. Preparar un archivo de firmas mutacionales (por ejemplo, generado con SigProfilerMatrixGenerator). 2. Ejecutar SigProfilerTopography con el archivo de entrada y especificar el genoma de referencia. 3. Analizar los resultados en el directorio de salida, que incluye mapas de asociación y estadísticas.

Errores comunes y buenas prácticas

  • Archivo de entrada mal formateado: Asegúrate de que el archivo CSV de firmas tenga las columnas correctas (por ejemplo, mutaciones, conteos, etc.).
  • Genoma de referencia incorrecto: Usa "hg38" o "hg19" según tu conjunto de datos.
  • Falta de permisos: Asegúrate de tener permisos de escritura en el directorio de salida.
  • Memoria insuficiente: Para grandes datos, considera reducir el tamaño de la región analizada o usar un sistema con más RAM.
  • Actualización del paquete: Mantén SigProfilerTopography actualizado para aprovechar mejoras y correcciones.
